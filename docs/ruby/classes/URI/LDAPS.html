Class URI::LDAPS < URI::LDAP

  • ruby/lib/uri/ldaps.rb

The default port for LDAPS URIs is 636, and the scheme is ‘ldaps:’ rather than ‘ldap:’. Other than that, LDAPS URIs are identical to LDAP URIs; see URI::LDAP.

Constants

DEFAULT_PORT = 636
