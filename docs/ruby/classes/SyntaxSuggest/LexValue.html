Class SyntaxSuggest::LexValue < Object

  • ruby/lib/syntax_suggest/lex_value.rb

Value object for accessing lex values

This lex:

[1, 0], :on_ident, "describe", CMDARG

Would translate into:

lex.line # => 1
lex.type # => :on_indent
lex.token # => "describe"
