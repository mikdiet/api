Class Bundler::URI::HTTPS < Bundler::URI::HTTP

  • ruby/lib/bundler/vendor/uri/lib/uri/https.rb

The default port for HTTPS URIs is 443, and the scheme is 'https:' rather than 'http:'. Other than that, HTTPS URIs are identical to HTTP URIs; see Bundler::URI::HTTP.
