Class RSS::Atom::Entry < RSS::Element

  • ruby/lib/rss/atom.rb
  • ruby/lib/rss/dublincore/atom.rb

Defines a top-level Atom Entry element, used as the document element of a stand-alone Atom Entry Document. It has the following attributes:

  • author

  • category

  • categories

  • content

  • contributor

  • id

  • link

  • published

  • rights

  • source

  • summary

  • title

  • updated

def have_author?
  authors.any? {|author| !author.to_s.empty?} or
    (source and source.have_author?)
end
